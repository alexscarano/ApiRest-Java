# ApiRest-Java

Projeto de teste com implementação de uma API RESTful utilizando **Java**.

## Tecnologias Utilizadas

- **Java**

## Sobre

Este repositório contém exemplos para criar endpoints REST, manipular requisições e estruturar uma API básica em Java.

## Como executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/alexscarano/ApiRest-Java.git
   ```
2. Importe o projeto em uma IDE Java.
3. Siga as orientações das pastas/examples para rodar e testar a API.
